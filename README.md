# 极简计算器

一个使用原生 HTML / CSS / JavaScript 实现的计算器项目，支持点击与键盘输入两种交互方式。

## 功能特性

- 基础四则运算：`+`、`-`、`×`、`÷`
- 运算优先级：先乘除，后加减
- 连续表达式输入与计算（如 `8×9×9×9+3+3`）
- 连续按等号重复运算
- 百分号延迟转换（如 `9%` 先显示，按 `=` 后得到 `0.09`）
- 正负号切换（输入态负数显示为 `(-x)`，结果态显示为 `-x`）
- 退格删除（结果态也可退格）
- 小数输入与浮点误差优化显示
- 大数字自动千分位分隔（`,`）
- 显示区字号自适应 + 溢出渐隐层叠提示
- 双行显示：
  - 上方：历史表达式
  - 下方：当前输入/结果

## 键盘快捷键

- 数字：`0-9`
- 小数点：`.`
- 运算符：`+` `-` `*` `/`（`x` / `X` 也可表示乘法）
- 百分号：`%`（`p` / `P` 也可）
- 等号：`Enter` 或 `=`
- 退格：`Backspace`
- 清空：`Esc` 或 `C`
- 正负切换：`N`

## 本地运行

这是一个纯前端静态项目，无需安装依赖。

直接用浏览器打开 `index.html` 即可运行，或使用任意静态服务器：

```bash
# 任选其一
python3 -m http.server 8000
# 或
npx serve .
```

然后访问 `http://localhost:8000`（或命令输出的本地地址）。

## 项目结构

```text
calculator/
```
